| Storage Need                       | AWS Service                             | Description                                                       |
| ---------------------------------- | --------------------------------------- | ----------------------------------------------------------------- |
| **General File Storage**           | **Amazon S3**                           | Store files, documents, images, videos — highly durable, scalable |
| **Block Storage for EC2**          | **Amazon EBS**                          | Virtual hard drives for EC2 — like a disk attached to your server |
| **Network File System**            | **Amazon EFS**                          | Shared file system for multiple servers (like a NAS)              |
| **Relational Databases**           | **Amazon RDS**                          | Store structured data (MySQL, PostgreSQL, etc.)                   |
| **NoSQL / Key-Value Store**        | **Amazon DynamoDB**                     | For scalable, fast key-value or document-based data               |
| **Archive / Long-Term Storage**    | **Amazon S3 Glacier**                   | Very cheap, long-term backup or archive storage                   |
| **Secrets / Secure Credentials**   | **AWS Secrets Manager**                 | Store API keys, passwords securely with auto-rotation             |
| **App Configs / Small Parameters** | **AWS Systems Manager Parameter Store** | Securely store config data or environment variables               |
| **Container Images**               | **Amazon ECR**                          | Store Docker images for use in ECS/EKS                            |
| **Analytics / Logs / Big Data**    | **Amazon S3 / Redshift / Athena**       | Store logs, events, and analyze them                              |


| Feature              | **Cookies**                                                                  | **Cache**                                                               |
| -------------------- | ---------------------------------------------------------------------------- | ----------------------------------------------------------------------- |
| **Purpose**          | Store **user-specific data** (e.g., login, preferences, tracking info)       | Store **website resources** to reduce load time (e.g., images, CSS, JS) |
| **Data Stored**      | Small text data (key-value pairs)                                            | Static content files (HTML, images, scripts)                            |
| **Size Limit**       | \~4 KB per cookie                                                            | Can store **MBs to GBs** depending on browser                           |
| **Created By**       | Website/server via HTTP headers or JavaScript                                | Browser automatically or via cache-control headers                      |
| **Expires**          | Can be set to expire at a specific time                                      | Can expire or be cleared by browser settings or cache headers           |
| **Accessed By**      | Server and client (JavaScript)                                               | Browser only; not accessible via JavaScript                             |
| **Sent to Server?**  | ✅ Yes — cookies are sent with every HTTP request                            | ❌ No — cache is stored locally and not sent with requests              |
| **Security Concern** | Can be used for **tracking**, needs proper protection (`HttpOnly`, `Secure`) | Generally safe, but stale cache can cause bugs or outdated pages        |
